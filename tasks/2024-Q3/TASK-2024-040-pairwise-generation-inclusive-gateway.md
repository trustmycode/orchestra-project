## Описание
Улучшить `ProcessToScenarioGenerator` для обработки `Inclusive Gateway`. Вместо генерации всех возможных комбинаций исходящих путей, необходимо реализовать алгоритм попарного тестирования (Pairwise) для создания оптимального набора сценариев, покрывающего все парные взаимодействия веток.

## Ключевые шаги
1.  Исследовать и выбрать/реализовать Java-библиотеку для генерации Pairwise-комбинаций (например, AllPairs4J).
2.  Интегрировать этот алгоритм в логику обработки `Inclusive Gateway` в `ProcessToScenarioGenerator`.
3.  Добавить в UI (в модальном окне генерации) опцию "Оптимизировать для Inclusive Gateways (Pairwise)".

## Критерии приемки
-   При генерации сценариев для процесса с `Inclusive Gateway`, количество созданных сценариев значительно меньше, чем `2^N - 1`.
-   Сгенерированный набор сценариев покрывает все парные комбинации исходящих веток.
