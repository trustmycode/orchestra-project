## Описание
Добавить поддержку PlantUML в веб-интерфейс.

## Ключевые шаги
1.  В `ImportView` добавить вкладку или кнопку для загрузки `.puml` файлов.
2.  В `ProcessDetailView` добавить компонент для рендеринга PlantUML (можно использовать готовые библиотеки для рендеринга SVG из текста PUML или обращаться к внешнему сервису рендеринга, если допустимо, или рендерить на бэкенде).
    *   *Рекомендация:* Рендерить на клиенте с помощью JS-библиотеки или использовать `plantuml-server` (в docker-compose), чтобы не зависеть от внешних сервисов.

## Критерии приемки
-   Пользователь может загрузить PUML-файл.
-   Пользователь видит визуализацию диаграммы на странице процесса.
