## Описание

Добавить интерфейс управления секретами в раздел настроек.

## Ключевые шаги

1.  В `SettingsView.tsx` добавить вкладку "Secrets".

2.  Реализовать таблицу со списком секретов (Key, Description, Actions). Значение секрета не показывать.

3.  Реализовать диалог создания/редактирования секрета.

    - Поле `Key` (валидация: только латиница, цифры, подчеркивание).

    - Поле `Value` (тип `password`).

4.  Добавить кнопку "Copy Reference", которая копирует в буфер строку `{{secret.KEY}}` для удобной вставки в настройки профилей.

## Критерии приемки

- Пользователь может добавить, обновить и удалить секрет.
- UI подсказывает правильный формат использования секрета.
