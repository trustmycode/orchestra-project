## Описание
Реализовать механизм сопоставления BPMN-задач с API-эндпоинтами из `ProtocolSpec`.

## Ключевые шаги
1. Создать компонент `EndpointMatcher`.
2. Реализовать первую, самую надежную стратегию сопоставления: поиск по `operationId`, указанному в метаданных BPMN-задачи.
3. `ProcessToScenarioGenerator` должен использовать `EndpointMatcher` для каждой задачи в найденном пути, чтобы получить информацию об эндпоинте.

## Критерии приемки
- Генератор успешно находит эндпоинт в `ProtocolSpec` по `operationId` из BPMN-задачи.
- Если сопоставление не найдено, задача пропускается с соответствующим предупреждением.
