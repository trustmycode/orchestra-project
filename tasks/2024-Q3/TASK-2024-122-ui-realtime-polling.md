## Описание

Пользователи вынуждены обновлять страницу, чтобы увидеть прогресс теста. Необходимо внедрить авто-обновление.

## Ключевые шаги

1.  Модифицировать `TestRunView.tsx`.

2.  Реализовать рекурсивный `setTimeout` или `useInterval` для опроса API `getTestRun(id)`.

3.  Интервал опроса: 1000мс.

4.  Условие остановки: статус прогона равен `PASSED`, `FAILED`, `FAILED_STUCK` или `CANCELLED`.

## Критерии приемки

-   При запуске теста пользователь видит, как появляются новые шаги и меняются статусы без перезагрузки страницы.
-   Когда тест завершается, опросы прекращаются (проверить в Network tab браузера).
