## Описание
Доработать `ProcessToScenarioGenerator` для сбора и нормализации условий с `sequence flow`, исходящих из шлюзов. Для каждого сгенерированного пути должен формироваться "предикат пути" (`Path Predicate`) — набор ограничений на данные.

## Ключевые шаги
1.  Реализовать парсер для извлечения выражений из поля `conditionExpression` у `sequence flow`.
2.  При обходе графа накапливать эти условия в структурированном виде (например, `{"constraints": {"age": [{"op": ">=", "value": 18}]}}`).
3.  Передавать этот `Path Predicate` вместе с информацией о пути на следующие этапы (генерация данных).

## Критерии приемки
-   Для пути, проходящего через шлюз с условием, генератор корректно формирует объект с ограничениями.
