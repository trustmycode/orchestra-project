## Описание
Реализовать AI-функцию анализа отчетов через агента `sgr-agent-core`. Агент должен получать ID прогона, самостоятельно собирать информацию об ошибках с помощью инструментов и генерировать рекомендации.

## Ключевые шаги
1.  Добавить в `AgentToolbox` инструменты:
    *   `getTestRunReport(runId)`: Получение JSON-отчета.
    *   `getStepDetails(stepId)`: Получение логов и payload шага.
2.  Реализовать сервис `ReportAnalystService`, который запускает агента с целью: "Проанализируй прогон X и дай рекомендации".
3.  Интегрировать вызов этого сервиса в UI отчета.

## Критерии приемки
-   Пользователь видит осмысленные рекомендации по исправлению упавшего теста, сгенерированные агентом.
