## Описание
Реализовать в `ProcessToScenarioGenerator` обработку циклов в BPMN-схеме.

## Ключевые шаги
1.  Реализовать алгоритм обнаружения циклов в `controlFlowGraph`.
2.  При обнаружении цикла применить стратегию "0-1-N" (с N=2 по умолчанию):
    *   Сгенерировать сценарий, где цикл не выполняется ни разу.
    *   Сгенерировать сценарий, где цикл выполняется один раз.
    *   Сгенерировать сценарий, где цикл выполняется N раз.
3.  Для итеративных сценариев передать в `AiDataGenerator` последовательность ограничений, чтобы данные менялись на каждой итерации, обеспечивая выход из цикла.

## Критерии приемки
-   Для процесса с циклом согласования генерируется как минимум три сценария, покрывающих 0, 1 и 2 итерации.
