## Описание
Создать страницы для отображения списка `SuiteRun` и детального отчета по одному `SuiteRun`.

## Ключевые шаги
1. Создать компонент `SuiteRunListView` для отображения истории запусков наборов.
2. Создать компонент `SuiteRunDetailView`, который показывает сводную информацию и список всех `TestRun`, входящих в `SuiteRun`.
3. Добавить ссылки для перехода на детальные отчеты по каждому `TestRun`.

## Критерии приемки
- Пользователь может видеть историю запусков `ScenarioSuite`.
- Пользователь может открыть отчет по `SuiteRun` и увидеть результаты всех входящих в него сценариев.
