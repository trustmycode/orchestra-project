## Описание
Интегрировать фреймворк `sgr-agent-core` в проект `ai-service` и создать базовую конфигурацию для LLM-агента. Эта задача является обязательным предварительным условием для остальных AI-задач.

## Ключевые шаги
1. Добавить зависимость `sgr-agent-core` в `pom.xml` сервиса `ai-service`.
2. Создать Spring-конфигурацию с основным бином `SgrAgent`.
3. Настроить подключение агента к LLM через Spring AI и `OllamaChatClient`.
4. Реализовать проверочный эндпоинт, запускающий агента с простой целью, чтобы убедиться, что цепочка «LLM + инструменты» работает.

## Критерии приемки
- `ai-service` успешно стартует с новой зависимостью.
- Примерный эндпоинт демонстрирует успешный вызов агента и ответ от локальной LLM через Ollama.
