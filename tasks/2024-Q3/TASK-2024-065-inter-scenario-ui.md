## Описание
Реализовать в UI возможности для задания зависимостей между сценариями и настройки экспорта данных.

## Ключевые шаги
1. На странице `ScenarioSuiteDetailView` добавить инструменты для визуализации порядка выполнения сценариев (drag-and-drop или модальное окно настроек).
2. В `ScenarioBuilderView` добавить поле для `exportAs` и подсказки по использованию `{{suite.*}}` плейсхолдеров.
3. Улучшить автодополнение плейсхолдеров, добавив варианты из `SuiteRun.context`.

## Критерии приемки
- Пользователь может в UI указать, что один сценарий должен выполняться после другого.
- Пользователь может настроить экспорт данных из шага.
