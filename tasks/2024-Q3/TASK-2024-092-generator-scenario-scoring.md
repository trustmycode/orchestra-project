## Описание
Внедрить механизм скоринга для автоматического ранжирования сгенерированных сценариев по их предполагаемой важности или рискованности.

## Ключевые шаги
1.  **Backend:** В `ProcessToScenarioGenerator` после создания каждого `TestScenario` вычислять для него `score` на основе набора эвристик (количество шлюзов, наличие критичных задач, длина пути).
2.  Сохранять `score` в `TestScenario`.
3.  **Frontend:** На странице `ScenarioSuite` отображать `score` и предоставить возможность сортировки/фильтрации по нему.
4.  (Опционально) В UI генерации добавить опцию "Сгенерировать только топ-N самых важных сценариев".

## Критерии приемки
-   Сгенерированные сценарии имеют рассчитанный `score`.
-   Пользователь может в UI отсортировать сценарии по их важности.
