## Описание
Реализовать эндпоинт, который возвращает список участников (Participants) или дорожек (Lanes) для заданного процесса. Это необходимо для UI "Мастера генерации", чтобы пользователь мог вручную привязать каждую дорожку к конкретной спецификации OpenAPI.

## Ключевые шаги
1.  В `ProcessService` реализовать логику парсинга сохраненного BPMN/PUML (или использования уже распарсенного графа) для извлечения списка участников.
2.  Создать эндпоинт `GET /api/v1/processes/{processId}/participants`.
3.  Возвращать JSON вида `[{ "id": "Participant_1", "name": "Order Service" }, ...]`.

## Критерии приемки
-   Эндпоинт возвращает корректный список участников для загруженного BPMN-процесса.
