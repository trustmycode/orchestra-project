## Описание
Создать пользовательский интерфейс для запуска и конфигурации процесса генерации `ScenarioSuite`.

## Ключевые шаги
1. На странице списка процессов (`ProcessListView`) добавить кнопку "Сгенерировать Suite".
2. Реализовать модальное окно с формой, где пользователь указывает название Suite и привязывает сервисы (дорожки BPMN) к импортированным спецификациям OpenAPI.
3. Реализовать отправку запроса на `POST /scenario-suites/from-process`.
4. После успешного ответа перенаправить пользователя на страницу нового `ScenarioSuite`.

## Критерии приемки
- Пользователь может запустить генерацию сценариев через UI.
- Пользователь может выбрать, какую спецификацию использовать для какой роли в процессе.
